(1) 개인정보 수집 유효기간

 * 유효기간이 지난 개인정보를 파기해야된다고 할때, 오늘 날짜로 파기해야 할 개인정보 번호들을 구한다.
 * 예 1) 예를 들어, A라는 약관의 유효기간이 12 달이고,
 2021년 1월 5일에 수집된 개인정보가 A약관으로 수집되었다면
 해당 개인정보는 2022년 1월 4일까지 보관 가능하며 2022년 1월 5일부터 파기해야 할 개인정보입니다.


(2) 조건
1.모든 달은 28일까지 있다고 가정합니다.
2. 변수
String today : 친구들 이름
String[] terms: 약관의 유효기간
String[] privacies : 개인정보의 정보
